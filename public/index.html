Aeternity Laboratory

The AEternity lab is a set of tools that enables people to try out and learn about the AEternity network.

The laboratory can build transactions, sign them, and submit them to the network. By default, AEternity test network is selected. However, user can choose to use his private network

User can create cryptographic keypairs.

These keypairs can be used on the Aeternity network where one is required.
