Aeternity Labs.

The aeternity lab is a set of tools that enables people to try out and learn about the aeternity network.

The laboratory can build transactions, sign them, and submit them to the network. It can also make requests to any of the Horizon endpoints.

These keypairs can be used on the Aeternity network where one is required.

For example, it can be used as an account master key, account signer, and/or as a aeternity-core node key.
